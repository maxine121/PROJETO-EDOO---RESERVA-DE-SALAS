Criamos um sistema de reservas de salas baseado em linguagem C++. O programa utiliza códigos com diferentes funcionalidades, com arquivos CPP, os HEADERS e os arquivos .txt. Utilizamos classes e heranças como foi solicitado no trabalho. Nosso objetivo nesse projeto foi melhorar a experiência do usuário quanto ao acesso de um sistema simples, prático e intuitivo para reservar salas de modo eficiente na universidade.

Para rodar o código basta cloná-lo no Clion e mover os documentos .txt para a pasta cmake-build-debug.
